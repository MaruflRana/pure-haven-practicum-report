# Pure Haven BD Practicum Report

Repository for the practicum report titled “Design and Development of a Web-Based E-Commerce Management System for Pure Haven BD.”

This repository follows the IUBAT BCSE practicum report structure.

Main actors: Customer/Visitor and Admin.

Important report rules:

- Do not use canteen-system content.
- Do not claim customer registration/login.
- Do not claim real payment gateway API integration.
- Payment should be described as payment method selection supporting Cash on Delivery, bKash, and Nagad.

Report artifacts include chapters, figures, tables, front matter, references, and final DOCX files.

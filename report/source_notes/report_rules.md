# Report Rules

- Use project facts from the approved planning sheet.
- Write in formal IUBAT BCSE practicum style.
- Use placeholders like [not available] for missing personal/organization data.
- Do not expose internal code risks or warnings in the final practicum report.
- Keep diagrams black and white, font size around 11, no overlapping, no unnecessary actors.
- Use “Customer/Visitor,” not “Registered User.”
- Use “Admin,” not “Super Admin” unless confirmed.
